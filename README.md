Файл **main.py** є головним запусковим файлом настільного застосунку, створеного на Tkinter.
Саме він ініціалізує інтерфейс, підключення до бази даних та всі основні модулі програми.
Для старту застосунку достатньо виконати команду:

```
python main.py
```

Після цього відкривається головне вікно програми.
